# LyX + XeLaTeX で日本語文書作成する私的テンプレート
LyX 2.3.x で動作させる前提.
`bxjsarticle`, `beamer` 用のテンプレート.

* ZXjatype に依存
http://zrbabbler.sp.land.to/zxjatype.html
* コンパイルには `-shell-escape` が必要

# 使い方
Linux 系なら
```
~/.lyx/templates
```
Mac なら
```
~/Library/Application Support/Lyx/<VERSION>/templates
```
に .lyx を配置する.
